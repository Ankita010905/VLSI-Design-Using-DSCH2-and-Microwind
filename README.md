# VLSI Design Using DSCH2 & Microwind 3.1

## Overview

This repository contains various digital and CMOS circuit designs implemented using DSCH2 and Microwind 3.1 as part of VLSI and Digital System Design studies.

The repository covers schematic design, logic verification, CMOS layout design, and simulation of fundamental digital circuits.

- Microwind 3.1
