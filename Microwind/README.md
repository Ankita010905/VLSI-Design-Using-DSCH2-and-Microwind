# 🔷 Microwind CMOS Layout Designs

This folder contains CMOS layout designs created using **Microwind 3.1**. The layouts demonstrate transistor-level implementation and physical design of CMOS circuits.

---

## 📋 Layouts Implemented

### MOS Devices
- NMOS Transistor
- PMOS Transistor

### CMOS Logic Circuits
- CMOS Inverter
- NAND Gate
- NOR Gate
- XOR Gate
- Other CMOS Logic Gates

### Oscillator Design
- Ring Oscillator using 5 CMOS Inverters

---

## 🎯 Objectives

- Learn CMOS layout design methodology.
- Understand transistor-level implementation.
- Perform layout generation and verification.
- Explore physical design concepts in VLSI.
