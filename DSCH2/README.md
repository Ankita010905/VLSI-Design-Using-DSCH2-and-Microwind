# 🔷 DSCH2 Digital Circuit Designs

This folder contains digital circuit designs developed and simulated using **DSCH2**. The designs cover fundamental combinational and sequential circuits used in Digital Electronics and VLSI Design.

---

## 📋 Circuits Implemented

### Basic CMOS Circuits
- CMOS Inverter
- Logic Gates
  - AND Gate
  - OR Gate
  - NAND Gate
  - NOR Gate
  - XOR Gate
  - XNOR Gate

### Combinational Circuits
- Full Adder
- Full Subtractor
- Binary Parallel Adder
- Multiplexer (MUX)
- Multiplier

### Sequential Circuits
- Different Types of Registers
- Asynchronous 4-Bit Up/Down Counter
- Decade Counter
- Digital Clock

### Special Circuits
- Seven Segment Display
- Transmission Gate
- Tri-State Inverter
